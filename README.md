## Installation
```
$ git clone https://github.com/SamuelCadau/MyGKrellm.git
```

### Compilation
Utilisé les fonctionnalités du Makefile comme make ou make re

```
$ make
```

### Comment utilisé ?

Exécutez le binaire

```
$ ./MyGKrellm
```

ou avec l'interface graphique ncurse

```
$ ./MyGKrellm -t
```
ou avec l'interface graphique SFML

```
$ ./MyGKrellm -g
```
